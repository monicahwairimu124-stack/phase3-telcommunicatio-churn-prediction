# Telecom Customer Churn Prediction

## Project Overview
Predicting customer churn for a telecommunications company using machine learning.

## Project Structure
- `notebooks/` - Jupyter notebooks for analysis and modeling


## Dataset
The dataset `bigml_59c28831336c6604c800002a.csv` contains customer information and churn status.


## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Jupyter Notebook
