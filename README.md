# Telecom Customer Churn Prediction

## Project Overview
Predicting customer churn for a telecommunications company using machine learning.

## Project Structure
- `notebooks/` - Jupyter notebooks for analysis and modeling
- `data/` - Dataset files (not tracked in Git)
- `src/` - Source code modules (optional)
- `requirements.txt` - Python dependencies (optional)

## Dataset
The dataset `bigml_59c28831336c6604c800002a.csv` contains customer information and churn status.

## Usage
1. Clone the repository
2. Place the dataset in the `data/` folder
3. Install dependencies: `pip install -r requirements.txt`
4. Run the analysis: Open `notebooks/churn.ipynb`
